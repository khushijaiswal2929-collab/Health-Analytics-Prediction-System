
# Health Analytics & Prediction System using CDC BRFSS Dataset

## Overview

This project focuses on healthcare analytics and predictive modeling using the CDC BRFSS dataset to identify populations with high healthcare-support needs. The system applies data preprocessing, exploratory data analysis (EDA), feature engineering, clustering, and machine learning techniques to generate meaningful healthcare insights.

The project analyzes demographic, lifestyle, and health-related data to uncover trends and predict individuals requiring increased healthcare support.

---

## Dataset

The project uses the CDC BRFSS dataset, which contains survey data related to:

* Demographics
* Physical & mental health
* Lifestyle behaviors
* Chronic diseases
* Healthcare access

The dataset required preprocessing to handle missing and invalid survey response values.

---

## Exploratory Data Analysis (EDA)

EDA was performed to analyze healthcare trends and population behavior patterns. The analysis included:

* Age and income distribution
* State-wise respondent analysis
* Smoking habit trends
* Chronic condition analysis

Visualizations were created using Matplotlib to better understand healthcare patterns and support data-driven insights.

---

## Machine Learning & Analytics

The project includes:

* Data preprocessing & feature engineering
* Creation of the `HighSupportNeed` target variable
* Population segmentation using clustering
* Predictive modeling using Logistic Regression and tree-based algorithms
* Model evaluation using accuracy, precision, recall, and confusion matrix
